Purpose of TaskFlow

TaskFlow is a lightweight task-management web application designed to help users organize tasks, view priorities, and track completion status.

How to Use TaskFlow

  1. Open the Home page to view task summary counts.
  2. Use the Add Task page to enter a new task.
  3. Use the Task List page to view all available tasks.
  4. Use the Task Detail page to review full information about one task.
  5. Use the navigation bar to move between all pages.

CI/CD Learning Goal

This project demonstrates branching, pull requests, Docker image building, Docker Hub publishing, and automated deployment using GitHub Actions.
